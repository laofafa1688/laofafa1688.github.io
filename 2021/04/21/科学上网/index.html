科学上网

2021-04-21 13:45:33

1.下载代理工具

  1. 官网下载地址
  2. 点击 v2rayN-Core.zip 下载
  3. 下载后解压到文件夹
  4. 点击 v2rayN.exe 开始运行

2.注册链接

  1. 火星云T1注册地址
  2. 输入账号 邮箱 密码注册
  3. 开始免费试用（每日签到可获取流量）
  4. 点击首页，点击 复制V2Ray订阅链接

    注册邮箱 微软outlook免费注册 可不用手机号免费注册

3.配置代理

  1. 打开 v2rayN
  2. 点击订阅 - 订阅设置 - 粘贴刚刚复制的链接 到 地址（URl） - 点击确认
  3. 点击订阅 - 更新订阅，开始更新下载服务器地址
  4. Ctrl + A 全选中所有服务器 Ctrl + R 测试真链接延迟
  5. 选中延迟低的 按 Enter 设为活动服务器
  6. 右击右下角任务栏 v2rayN 图标，系统代理 - 自动配置系统代理
  7. 打开 Google，等墙外网站 即可正常访问！

暗网？

  1. 点击下载Tor洋葱浏览器
  2. 配置 v2rayN ，打开v2rayN - 设置 - 参数设置 - 关掉流量侦测 - 复制本地端口
  3. 下载完Tor后双击安装 - 运行 - 配置 - 代理 - 选择Socket5 - 输入 127.0.0.1 端口粘贴 v2rayN 端口，
  4. 点击连接，等待连接成功即可，
  5. 使用Tor 开始访问 onion 网络
  6. 开始访问暗网，打开 https://github.com/sunfall-cell/- 复制地址，在Tor 浏览器中打开浏览即可
